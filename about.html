About This Quran Project

Date Created: 26 April 2025

Created By: Mussa Omar Khamis

This online Quran project was developed to provide an easy and peaceful way to access the Holy Quran using a simple web interface. It was created with the goal of helping Muslims read and reflect on the words of Allah at any time, from any place.

The website includes the original Arabic text of the Quran. It is designed to be lightweight, fast, and accessible, with no ads or distractions.

This project was built using HTML, CSS, and JavaScript, and it uses Supabase as the backend for handling features such as views, likes, and comments.

May Allah accept this effort and make it a source of benefit and guidance for all who visit.
